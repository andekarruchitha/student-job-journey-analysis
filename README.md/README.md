# 🎓 Student to Job Journey Analysis

A data analytics project that explores how student performance factors like CGPA, internships, attendance, certifications, and communication skills impact job placement and salary outcomes.

---

## 📌 Project Overview

This project analyzes student data to understand the relationship between academic performance and career success. It helps identify key factors that influence:
- Job placement chances
- Salary packages
- Skill importance in hiring

---

## 📊 Features

- Placement distribution analysis
- CGPA vs Salary insights
- Internship impact on salary
- Communication skills analysis
- Data visualization using charts

---

## 🛠️ Technologies Used

- Python 🐍
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📈 Key Insights

- Higher CGPA students generally get better salary packages
- Internships improve placement chances significantly
- Communication skills play an important role in selection
- Students with multiple certifications perform better
